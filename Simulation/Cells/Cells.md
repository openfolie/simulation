A single block in the grid we make for the environment

An agent will be able to:
- observe (current & neighbouring cells)
- modify (current cell)

current cell means the cell agent is on top of

There are a lot of properties inside a cell:
- Climate
- Elevation
- Terrain

There are a few computed properties
- Humidity: Derived from the current state of [[WaterCycle|Water Cycle]]
- Temperature: Derived from the current state of [[WaterCycle|Water Cycle]]
